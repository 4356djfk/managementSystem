# 前端页面与功能报告

## 1. 页面总览

当前前端主要页面如下：

1. 登录页 `/login`
2. 注册页 `/register`
3. 仪表盘 `/dashboard`
4. 模板中心 `/dashboard/templates`
5. 用户管理 `/dashboard/users`
6. 我的日历 `/dashboard/calendar`
7. 全局搜索 `/dashboard/search`
8. 审计日志 `/dashboard/audit`
9. 通知中心 `/dashboard/notifications`
10. 个人资料 `/dashboard/profile`
11. 项目编辑器 `/editor/:projectId`

## 2. 页面功能说明

### 2.1 登录页

功能：

1. 用户登录
2. 登录后跳转原目标页
3. 会话失效后自动回流登录页

### 2.2 注册页

功能：

1. 新用户注册
2. 与后端注册接口联调

### 2.3 仪表盘

功能：

1. 项目列表与筛选
2. 项目详情弹窗
3. 新建项目
4. 从模板建项目
5. 初始化演示项目
6. 待办任务与当日工时提醒
7. 首页统计卡片与里程碑提醒

### 2.4 模板中心

功能：

1. 项目模板查看
2. 模板新增与编辑

### 2.5 用户管理

功能：

1. 用户查询
2. 用户新增
3. 用户编辑
4. 用户状态变更
5. 用户角色调整

### 2.6 我的日历

功能：

1. 查看与本人相关的日历事件

### 2.7 全局搜索

功能：

1. 关键字搜索
2. 按类型和项目过滤

### 2.8 审计日志

功能：

1. 查看系统操作记录
2. 按条件过滤

### 2.9 通知中心

功能：

1. 查看通知
2. 标记已读
3. 全部已读

### 2.10 个人资料

功能：

1. 查看当前账号信息
2. 编辑个人资料

### 2.11 项目编辑器

功能：

1. 项目概览
2. 项目章程
3. 项目成员
4. WBS
5. 需求
6. 范围基线
7. 任务与评论
8. 里程碑
9. 任务依赖
10. 甘特图
11. 关键路径
12. 进度预警
13. 风险与风险矩阵
14. 成本计划 / 基线 / 实际成本 / EVM
15. 质量计划 / 活动 / 指标
16. 测试计划 / 用例 / 缺陷 / 测试报告
17. 配置项 / 配置基线
18. 变更请求与审批日志
19. 沟通矩阵 / 会议 / 沟通记录
20. 供应商 / 采购 / 合同
21. 工时 / 工时报表
22. 验收项 / 归档项 / 经验教训
23. 周报 / 月报 / 总结报告
24. 附件上传、下载、删除
25. 导入导出

## 4. 当前结论

从页面覆盖率看，前端已经具备完整项目管理系统的主流程，不再属于“只做了首页和少量 CRUD”的阶段。
